# Job Portal Application

A full-stack Job Portal web application that connects job seekers with employers. Users can create accounts, browse jobs, apply for positions, and manage applications, while companies can post and manage job listings.

## Features

### User Features
- User Registration & Login
- Secure Authentication
- Browse Available Jobs
- Apply for Jobs
- Manage Profile
- Track Applications

### Company Features
- Company Registration
- Create & Manage Job Listings
- View Applicants
- Upload Company Logo
- Dashboard Analytics

## Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)

### Cloud Storage
- Cloudinary
